📄 AI Resume Analyzer

An intelligent AI-powered Resume Analyzer built using Machine Learning + NLP that evaluates resumes, predicts job categories, recommends roles, and calculates an ATS (Applicant Tracking System) score.

🔗 Live App:
👉 https://ai-resume-analyzer-ad2hv6jzu5qvpszewbfmg7.streamlit.app/

The trained ML models used in this project are hosted on Hugging Face:

👉 https://huggingface.co/Subh737/resume-analyzer-model

🚀 Features

✅ Resume upload (PDF format)
✅ Resume text extraction
✅ Resume category prediction
✅ Job role recommendation
✅ ATS score calculation
✅ Skills & keyword analysis
✅ Clean Streamlit web interface
✅ Cloud-hosted ML models (Hugging Face)

🧠 How It Works

User uploads a resume (PDF)
Text is extracted and cleaned
TF-IDF vectorizers transform resume text
Random Forest models predict:
Resume Category
Job Recommendation
ATS score is calculated based on keyword matching
Results displayed instantly in the web app

🏗️ Tech Stack

Frontend: Streamlit
Backend: Python
Machine Learning: Scikit-learn
NLP: spaCy
Model Hosting: Hugging Face Hub
Deployment: Streamlit Cloud
Version Control: GitHub
